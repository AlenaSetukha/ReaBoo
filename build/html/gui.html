Application components¶

This section describes integral interface components of application.

Provides GUI for flesh-card learning app.

class LanguageBar - top bar widget for main window; class MainWindow - first screen for user to see; class DictionaryWindow - screen for amending user’s dictionary; class NewCard - widget for creating and amending words in dictionary; class MyScreenManager - screen stack controller; class ReaBooApp - the application itself.
